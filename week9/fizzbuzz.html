Fizz Buzz

Write a JavaScript function that prints numbers from 1 to 20. However:

  • If the number is divisible by 3, print "Fizz"
  • If the number is divisible by 5, print "Buzz"
  • If the number is divisible by both 3 and 5, print "FizzBuzz"
  • Otherwise, print the number itself.
  • Steps to complete the task.
  • Use a loop (for loop) to iterate from 1 to 20.
  • Use conditional statements (if-else) to check divisibility.
  • Print the correct output based on the conditions.
